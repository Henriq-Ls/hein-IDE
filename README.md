⚠️ Aviso do Windows — Leia antes de instalar
Por ser um software novo sem certificado digital, o Windows vai exibir dois avisos. Isso é normal e esperado. Siga os passos abaixo:

Passo 1 — Na hora do download (navegador)
O navegador pode bloquear o arquivo com a mensagem "hein-IDE.exe normalmente não é baixado".

➡️ Clique nos três pontinhos (...) ao lado do aviso → "Manter" ou "Manter assim mesmo"

Passo 2 — Na hora de abrir (Windows Defender)
Ao tentar abrir o .exe, o Windows pode exibir:

"O Windows protegeu o computador — O Microsoft Defender SmartScreen impediu que um aplicativo não reconhecido fosse iniciado."

➡️ Clique em "Mais informações" → depois em "Executar assim mesmo"
